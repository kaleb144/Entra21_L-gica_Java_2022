## Aprendendo sobre operadores realacionais

## Tipos:

> (<=) menor que (essa comparação não verifica se o número é igual também)
>
> (<=) menor ou igual que (essa comaparação verifica se o número é igual)
>
> (>)  maior que ( essa comparação não verifica se o número é igual também)
>
> (>=) maior ou igual que (essa comparação verifica se o número também é igual)
